# EzyBIM Official Brand Kit

This brand kit consolidates the visual identity used in the EzyBIM website for consistent use in software UI, Revit add-ins, and other marketing materials.

## 🎨 Color Palette

### Primary Colors
Used for core actions, branding highlights, and key interactive elements.
- **Primary Teal**: `#33B5B5` (The main brand color)
- **Primary Containers**: `#33B5B5` (at 10-20% opacity for backgrounds)
- **On-Primary**: `#FFFFFF` (White text on teal backgrounds)

### Foundation Colors
Used for the overall structure, sidebars, and high-readability text.
- **Slate Foundation**: `#1E293B` (Deep Navy/Slate - Used for Footers, Dark buttons)
- **Secondary Slate**: `#2C3E50` (Used for headers and main body text)
- **Surface (Background)**: `#FBFCFE` (Very light blue-ish white)

### Semantic & Accents
- **Surface Container**: `#F0F4F8`
- **Surface High**: `#E2E8F0`
- **Surface Highest**: `#CBD5E1`
- **Outline Variant**: `#94A3B8`
- **Slate Text (Variant)**: `#64748B` (Muted body text)

---

## 🔡 Typography

### Headlines (Display)
- **Family**: `Manrope`, ui-sans-serif, system-ui, sans-serif
- **Weights**: 700 (Bold), 800 (Extra Bold)
- **Style**: Tight letter-spacing (`tracking-tighter`), used for large titles.

### Body Text
- **Family**: `Inter`, ui-sans-serif, system-ui, sans-serif
- **Weights**: 300 (Light), 400 (Regular), 600 (Semi-bold)
- **Style**: Clean, accessible, and high-readability.

---

## ✨ Visual Language

### The "BIM Grid"
A signature visual element representing precision and architectural modeling.
- **Pattern**: 40px x 40px square grid.
- **Line Color**: `rgba(51, 181, 181, 0.1)` (10% Primary Teal).
- **Usage**: Background overlays for Hero sections and major UI containers.

### Glassmorphism
Used for "Active" states, project cards, and floating panels.
- **Background**: `rgba(255, 255, 255, 0.7)` (70% White).
- **Blur**: `16px` backdrop filter.
- **Border**: `1px solid rgba(255, 255, 255, 0.2)`.

### Corner Radius
- **Default**: `0.125rem` (Slightly rounded for a technical look).
- **Large**: `0.25rem`.
- **Extra Large**: `0.5rem`.
- **Full Shell/Pill**: `9999px`.

## 🖼️ Logo Assets
- **Logo Full**: Icon + Wordmark (Navbar)
- **Logo Compact**: Icon + Wordmark tight (Footer)
- **Logo Icon**: "E" Graphic only (Favicon, App Launcher)

## 💡 UI Implementation Tips for Tools
- **Buttons**: Use the **CTA Gradient** (`#33B5B5` to `#008080`) for the primary action.
- **Micro-Animations**: Use smooth, eased transitions (0.3s - 0.5s duration) for hover states.
- **Contrast**: Always ensure active text has high contrast against background slates or teals.
